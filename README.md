Краткое описание проекта:
	- Регистрация и вход пользователей в систему
	- Подразделение уровней доступа на ученика, повара и администратора
	- Покупка блюд и оформление абонемента
	- Получение уведомлений
	- Написание отзывов о блюдах
	- Отправка заявок на закупку
	- Добавление блюд в меню
	- Указание пищевых аллергий
	- Управление количеством блюд в наличии
	- Управление учатсниками
	- Удаление отзывов
	- Создание отчетов
	- Подтверждение заявок
	


Инструкция по установке/развертыванию:
	Есть несколько вариантов установки:
		1. Просмотр функционала по ссылке: https://school-cafeteria-6882.vercel.app
		2. Скачать архив с GitHub 
		   -> Распаковать его в любое удобное место на вашем компьютере 
		   -> Установить Node JS с официального сайта (https://nodejs.org/en/download)
		   -> Перейти в ранее скачанную папку и запустить .bat файл project\INSTALLATION.bat 
		   -> Ожидать установки и запуска браузера(3-5 минут)
